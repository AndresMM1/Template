# Clean Project Template

This is a fresh project template based on your Incident Studio stack.

## Included
- **Vite + React + TypeScript**
- **Tailwind CSS** (configured with your custom theme)
- **Shadcn UI Components** (`src/components/ui`)
- **Routing** (`react-router-dom`)
- **Utils** (`cn` helper)

## Getting Started

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Development Server:**
    ```bash
    npm run dev
    ```

3.  **Start Building:**
    -   Edit `src/pages/Home.tsx` or add new pages in `src/pages`.
    -   Update `src/App.tsx` to add routes.
